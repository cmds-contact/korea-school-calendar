# 한국 학교 학사일정 가이드 (2025학년도)

한국의 초·중·고·대학교 학기 및 방학 일정을 한눈에 파악할 수 있는 웹사이트입니다.

## 용도

교육 사업 프로그램 기획 시 학생들의 방학 일정에 맞춰 프로그램을 기획하기 위한 참고 자료로 활용할 수 있습니다.

## 주요 기능

### 1. 프로그램 기획 캘린더 (`planner.html`)

- **연간 타임라인 시각화**: 3월~다음해 2월까지 학기/방학을 가로 바 차트로 표시
- **학교급 필터**: 체크박스로 초/중/고/대학 표시 ON/OFF
- **방학 기간 강조**: 프로그램 기획 타겟 시기를 시각적으로 구분
- **방학 기간 요약표**: 필터와 연동되는 요약 정보

### 2. 상세 정보 페이지 (`index.html`)

- 학교급별 전형적인 일정 (전국 기준)
- 서울/경기 2025학년도 실제 예시 날짜
- 학교급별 특징 및 참고사항

## 2025학년도 방학 기간 요약

### 여름방학
| 학교급 | 기간 | 일수 |
|--------|------|------|
| 초등학교 | 7/19~25 ~ 8/18~25 | 약 4주 (21~35일) |
| 중학교 | 7/17 ~ 8/18 | 약 32일 |
| 고등학교 | 7/15~20 ~ 8/19~26 | 약 4~6주 |
| 대학교 | 6/17~24 ~ 8월말 | 약 70일 |

### 겨울방학
| 학교급 | 기간 | 일수 |
|--------|------|------|
| 초등학교 | 12/22 ~ 2월말 | 약 2개월 (30~60일) |
| 중학교 | 12/27 ~ 2월초 | 약 30~40일 |
| 고등학교 | 12/20~31 ~ 2월 | 약 1~2개월 |
| 대학교 | 12/22 ~ 2/28 | 약 2개월+ |

## 사용 방법

1. `planner.html`을 브라우저에서 열기
2. 상단 체크박스로 보고 싶은 학교급 선택
3. 타임라인과 요약표에서 방학 기간 확인
4. 상세 정보가 필요하면 `index.html`로 이동

## 파일 구조

```
korea-school-calendar/
├── README.md           # 프로젝트 문서
├── index.html          # 상세 정보 페이지
└── planner.html        # 프로그램 기획용 타임라인 대시보드
```

## 기술 스택

- HTML5
- CSS3 (인라인 스타일)
- Vanilla JavaScript
- 반응형 디자인 (모바일/데스크톱 호환)

## 데이터 출처

- 서울/경기 교육청 학사일정

- 2025학년도 기준 (2025.3 ~ 2026.2)

## 주의사항

실제 날짜는 학교별로 1~3일 정도 차이가 날 수 있습니다.
프로그램 기획 시 각 학교의 공식 학사일정을 반드시 확인하세요.
